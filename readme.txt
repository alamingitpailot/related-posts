Displays related posts below content based on categories to enhance engagement and navigation.

== Description ==

By dynamically displaying related posts below the main content, the RelatedPosts plugin improves user engagement and navigation on your WordPress website. This plugin provides users with relevant content by intelligently identifying and displaying posts from the same category as the current post.

= Feedback = 
This plugin is actively being developed & improved. We’re far from done and are adding new features, functionality, and fixes daily. If there’s a feature you want but we don’t have yet, please [let us know](mailto:alamincmt7418@gmail.com "Send feedback").


 
== Installation ==

**Uploading in WordPress Dashboard**

* Download related-posts.zip
* Navigate to the ‘Add New’ in the plugins dashboard
* Navigate to the ‘Upload’ area
* Select related-posts.zip from your computer
* Click ‘Install Now’
* Activate the plugin in the Plugin dashboard
